Entrada:
- N bits no argumento 1
- N bits no argumento 2
- T total de operações
- M bits para a operação ($m = \log_{2}{T})$

Saída:
- N bits no resultado da operação
- Bits de flags (1 bit para cada flag):
	- Zero (Z)
	- Negativo (N)
	- Overflow (V)
	- Carry out (C)
